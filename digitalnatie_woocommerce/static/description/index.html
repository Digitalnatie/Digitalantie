Data flows

What syncs — in both directions

▼ WooCommerce → Odoo (import)

  • • Products and variants — categories, images, tags, taxes
  • • Customers — billing & shipping addresses
  • • Orders — lines, shipping, fees, discounts, taxes
  • • Stock recovery import (manual, on-demand)
  • • Price recovery import (manual, on-demand)
  • • Custom WooCommerce meta_data → Odoo fields

▲ Odoo → WooCommerce (export)

  • • Products, variants, descriptions, images, prices
  • • Publication state — draft / published
  • • Stock levels — Odoo is the single source of truth
  • • Order fulfillment status updates
  • • Custom Odoo field values → meta_data
  • • Full and partial refunds via refund wizard

Getting started

Up and running in 5 steps

1

Drop the module in your addons folder

Copy digitalnatie_woocommerce into any directory declared in your addons_path.

2

Update apps list & install

Activate developer mode → Apps → Update Apps List → install WooCommerce Connector.

3

Generate WooCommerce REST API keys

WooCommerce → Settings → Advanced → REST API → Add Key → Read/Write → copy Consumer Key & Secret.

4

Create your first instance in Odoo

WooCommerce → Instances → New → enter URL, keys and warehouse → click Test Connection.

5

(Optional) Register webhooks & activate crons

Click Register Webhooks on the instance form. Enable crons under Settings → Technical → Scheduled Actions.

FAQ

Common questions

QDoes this require a license server or activation key?

No. The module is fully self-contained. No telemetry, no SaaS, no key server. Standard OPL-1 conditions apply.

QWill scheduled syncs overwrite my Odoo stock?

Never. Export Inventory always pushes Odoo → WooCommerce. To pull stock back, use Import Stock (Recovery) — gated by a confirmation dialog.

QCan I connect multiple WooCommerce stores?

Yes. Create one instance record per store. Crons iterate every active instance and isolate errors per store.

QWhich WooCommerce versions are supported?

Any version exposing the wc/v3 REST API — WooCommerce 3.5 and later.
